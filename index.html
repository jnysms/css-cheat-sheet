✿ CSS Cheat Sheet ✿

Element Description Example
font-family: Sets font family for section. Cambria, Cochin, Georgia, Times, 'Times New Roman', serif;
font-size: Sets text size. 18px; -or- 3em;
font-weight: Sets the boldness of text. Bold; normal; 400; lighter;
font-style: Sets the style of the text. Italic; normal;
color: Sets font color (foreground). Black; red; purple; #666;
background-color: Sets background color. Black; red; purple; #666;
text-decoration: Sets variety of effects on text. Underline; overline;
text-transform: Sets formatting of text. Uppercase; lowercase; capitalize;
text-align: Sets text alignment. Center; right; left;
vertical-align: Sets content relative to baseline. Auto; baseline; text-top;
line-height: Sets spacing between lines. 1px; 2em;
letter-spacing: Sets spacing between letters. 1px; 2em;
text-indent: Sets an indent of the first line. 1px; 2em;
border: Sets border of element. 1px; 2em;
position: Sets elements in place on screen. Absolute; fixed; relative;
display: Sets how element is placed in the doc flow. Block; inline; none;
float: Sets stacking of element horizontally. Left;
z-index: Sets stacking order of elements. z-index of 1 is below z-index of 2.
